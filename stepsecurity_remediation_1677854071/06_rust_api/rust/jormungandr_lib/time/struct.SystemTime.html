pub struct SystemTime(_);
Expand description

time in seconds and nanoseconds since UNIX Epoch

The human readable formatting is ISO8601 compliant.

Example


let time = SystemTime::now();

println!("now: {}", time);
// now: 2019-06-17T18:17:20.417032+00:00
