pub struct PeerComms { /* private fields */ }
Expand description

State of the communication streams that a single peer connection polls for outbound data and commands.

Dropping a PeerComms instance results in the client-side connection to be closed if it was established, or all outbound subscription streams of a server-side connection to be closed.
